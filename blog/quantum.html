Classical Information

Stochastic Matrix

A stochastic matrix is a square matrix used to describe transitions in a system, where each element represents the probability of transitioning from one state to another. It is often used in Markov chains and other probabilistic models.

Each row of the matrix must sum to 1, as it represents the probability distribution of the next state given the current state. For example, a 3x3 stochastic matrix might look like this:


P = [0.5  0.2  0.3]
    [0.4  0.4  0.2]
    [0.3  0.3  0.4]

In this matrix, the first row indicates that the probability of transitioning from state 1 to state 1 is 0.5, to state 2 is 0.2, and to state 3 is 0.3.

Note: Stochastic matrices are often used to model systems that evolve in discrete time steps, where the state of the system depends probabilistically on its previous state.
